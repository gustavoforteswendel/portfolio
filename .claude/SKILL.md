# Premium Front-End Experience Skill

## Philosophy

You are not just writing HTML, CSS and JavaScript.

You are designing memorable digital experiences.

Every project must feel handcrafted, cinematic and intentional, never like a generic template generated by AI.

---

# Forbidden Patterns

Never create layouts that rely on:

* Fixed `max-width: 1200px` containers centered on the screen.
* Large repetitive `padding: 120px 0`.
* Equal spacing between every section.
* Bootstrap-like structures.
* Cookie-cutter hero sections.
* Generic cards placed in simple grids.
* Repetitive "title + paragraph + button" compositions.
* Obvious AI layouts with identical margins and spacing.

If a layout looks predictable after one scroll, redesign it.

---

# Composition

Treat every viewport as a design canvas.

Alternate constantly between:

* left aligned content
* centered compositions
* asymmetrical layouts
* overlapping elements
* editorial spacing
* oversized typography
* negative space
* floating components
* full-screen imagery
* split layouts
* layered compositions

Each section should feel unique while still belonging to the same visual language.

---

# Rhythm

Create visual rhythm.

Do not repeat identical spacing.

Examples:

* compact section
* huge breathing space
* immersive section
* minimalist section
* dense composition
* cinematic transition

Scrolling should feel like reading a premium magazine.

---

# Typography

Typography is a primary design element.

Use:

* scale contrast
* mixed weights
* variable sizes
* intentional hierarchy
* oversized headlines
* subtle captions

Avoid making every heading identical.

Typography should create emotion before decoration.

---

# Motion

Motion is part of the interface.

Prioritize:

* GSAP
* ScrollTrigger
* FLIP
* SplitText when appropriate
* smooth stagger animations
* parallax
* image reveals
* clip-path transitions
* mask animations
* subtle rotations
* transform-based animations

Never animate only opacity.

Everything should have physical intention.

---

# Three.js

Only use Three.js when it enhances storytelling.

Good examples:

* interactive ID card
* floating objects
* premium product showcase
* subtle depth
* cursor interaction
* reflections
* soft lighting

Never use Three.js as decoration.

Performance always comes first.

---

# Images

Images should not simply appear.

Reveal them through:

* masks
* scale transitions
* clip-path
* cinematic crops
* parallax movement
* layered entrances

Large imagery is preferred over many small cards.

---

# Scroll Experience

Scrolling must tell a story.

Possible interactions:

* horizontal sections
* pinned storytelling
* progressive reveals
* immersive transitions
* section transformations
* layout morphing

Avoid long pages composed only of stacked blocks.

---

# Components

Avoid repetitive reusable blocks.

Each major section should have its own identity.

Buttons, cards and containers can share a design language, but compositions should evolve throughout the experience.

---

# Colors

Prefer restrained palettes.

Use one dominant background with subtle accent colors.

Create hierarchy through contrast, light and spacing instead of excessive color variation.

---

# Premium Feeling

The experience should resemble studios such as:

* Awwwards winners
* Immersive digital agencies
* Luxury architecture websites
* Editorial experiences
* High-end product launches

Never resemble a generic SaaS landing page.

---

# Responsiveness

Desktop is the flagship experience.

Mobile should preserve quality while adapting interactions intelligently.

Do not blindly stack desktop layouts.

Recompose sections specifically for smaller screens.

---

# Code Quality

Keep code modular and organized.

Separate concerns clearly.

Prefer semantic HTML.

Optimize animations.

Avoid unnecessary dependencies.

Keep performance and maintainability as priorities.

---

# Final Rule

Whenever generating a new section, ask internally:

> "Would this be immediately recognizable as an AI-generated layout?"

If the answer is yes, redesign it until it feels handcrafted, intentional and premium.
